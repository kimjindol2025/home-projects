「 골든 시그널 (Golden Signals) 」
  1. Latency: 사용자가 경험하는 시간
  2. Traffic: 시스템이 처리하는 일의 양
  3. Errors: 잘못된 처리의 비율
  4. Saturation: 한계에 얼마나 가까운가

「 메트릭 vs 로그 vs 트레이스 」
  메트릭: "지금 CPU는 60%"
  로그: "요청 처리 중 타임아웃 발생"
  트레이스: "Request A가 Service 1→2→3을 거쳤다"

「 관측성이 해결하는 문제 」
  ✓ 무엇이 문제인가? (메트릭 기반 진단)
  ✓ 왜 문제인가? (로그 기반 분석)
  ✓ 어디서 시작되었나? (트레이스 기반 추적)
