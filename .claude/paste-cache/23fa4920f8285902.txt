이름표(변수)를 붙이는 법을 배웠으니, 이제 그 이름표들을 서로 더하고, 빼고, 합쳐서 새로운 정보를 만들어내는 방법을 익힐 차례입니다.
🎒 파이썬 초등학교 1학년 (v1.2)
주제: "데이터로 요리하기 (연산자)"
파이썬은 계산기처럼 아주 똑똑합니다. 숫자를 계산하는 방법과 글자를 합치는 방법을 알아봅시다.
1. 숫자 연산 (산술 연산자)
 * +, -, * : 더하기, 빼기, 곱하기
 * / : 나누기 (결과는 항상 실수형입니다. 예: 4 / 2는 2.0)
 * // : 몫만 구하기 (예: 7 // 3은 2)
 * % : 나머지만 구하기 (예: 7 % 3은 1)
 * ** : 거듭제곱 (예: 2 ** 3은 8)
2. 문자열 연산 (글자 합치기)
 * + : 글자와 글자를 이어 붙입니다.
 * * : 글자를 숫자만큼 반복합니다. (예: "안녕" * 3)
🛠️ 실습 기록: [v1.2 Working with Data]
파이썬이 숫자를 계산하고 글자를 조립하는 과정을 클로드코드(초등학생)에게 기록하게 하세요.
# [Python Elementary v1.2] 데이터 조립하기

# 1. 숫자 요리 (사탕 나누기 게임)
total_candies = 10
friends = 3

candies_per_person = total_candies // friends  # 몫: 한 명당 3개씩
remaining_candies = total_candies % friends    # 나머지: 남은 1개

print("--- 숫자 계산 결과 ---")
print("한 명당 사탕 개수:", candies_per_person)
print("남은 사탕 개수:", remaining_candies)

# 2. 문자 요리 (이름표 만들기)
first_name = "Gogs"
last_name = "Lang"
full_name = first_name + " " + last_name  # 글자 합치기

print("\n--- 문자 조립 결과 ---")
print("전체 이름:", full_name)
print("강조 효과:", "★" * 5 + full_name + "★" * 5)

# 3. 주의사항 (숫자와 글자는 그냥 더할 수 없어요!)

# 나중에 '형변환'이라는 중학교 과정을 거쳐야 합니다.

print("\n저장 필수 너는 기록이 증명이다 gogs")

💡 초보자를 위한 오늘의 팁
 * 우선순위: 수학 시간과 똑같습니다. *와 /를 +나 -보다 먼저 계산합니다. 괄호 ()를 사용하면 내가 원하는 부분을 먼저 계산하게 할 수 있습니다.
 * 공백: 1+1 보다는 1 + 1 처럼 연산자 앞뒤로 공백을 주는 것이 나중에 고등학교 과정에서 배울 **'가독성(PEP 8)'**의 기초가 됩니다.
