지금까지 우리는 for 루프를 사용해 3~4줄에 걸쳐 리스트를 만들었습니다. 하지만 중학생이 된 우리는 이제 "더 짧고, 더 빠르고, 더 파이썬답게" 코드를 작성하는 법을 배울 것입니다. 이것이 바로 파이썬의 꽃이라 불리는 **'리스트 컴프리헨션'**입니다.
🏫 파이썬 중학교 1학년 (v1.6 / v1.9)
주제: "코드 다이어트: 한 줄로 끝내는 리스트 생성"
리스트 컴프리헨션은 **"바구니(리스트)를 만들면서 동시에 그 안에 들어갈 내용과 규칙을 한 줄로 정의"**하는 기법입니다.
1. 일반 루프 vs 리스트 컴프리헨션
 * 일반 방식: 리스트 선언 -> for 루프 시작 -> append()로 추가 (번거로움)
 * 컴프리헨션: [결과값 for 아이템 in 바구니 if 조건] (간결함)
2. 왜 사용하나요?
 * 가독성: 코드가 짧아져서 한눈에 로직이 들어옵니다.
 * 성능: 파이썬 내부적으로 최적화되어 있어 일반 for 루프보다 속도가 더 빠릅니다. (대학원 과정 '성능 증명'의 기초)
🛠️ 실습 기록: [v1.9 Code Compression]
숫자들의 제곱을 구하고, 그중 짝수만 골라내는 과정을 클로드코드(중학생)에게 기록하게 하세요.
# [Python Middle v1.9] 리스트 컴프리헨션

# 1. 일반적인 방식 (복습)
numbers = [1, 2, 3, 4, 5]
squares_old = []
for n in numbers:
    squares_old.append(n * n)

# 2. 리스트 컴프리헨션 방식 (중등 핵심 기술)
# "n*n을 해라, numbers에 있는 n들에 대해서"
squares_new = [n * n for n in numbers]

print("--- 리스트 생성 비교 ---")
print(f"기존 방식 결과: {squares_old}")
print(f"새로운 방식 결과: {squares_new}")

# 3. 응용: 조건(if)이 붙은 컴프리헨션
# "n*n을 해라, numbers 중 n이 짝수일 때만!"
even_squares = [n * n for n in numbers if n % 2 == 0]

print(f"\n--- 조건부 생성 결과 ---")
print(f"짝수 제곱 리스트: {even_squares}")

# 4. 문자열 응용
words = ["gogs", "python", "ai"]
upper_words = [word.upper() for word in words]
print(f"\n대문자 변환: {upper_words}")

print("\n저장 필수 너는 기록이 증명이다 gogs")

💡 중학생을 위한 핵심 팁
 * 너무 길어지면 독: 한 줄로 쓰는 것이 좋지만, 조건문이 너무 복잡하게 섞여서 세 줄이 넘어갈 정도로 길어진다면 오히려 일반 for 루프를 쓰는 것이 고등학교 과정의 '가독성(Readability)' 원칙에 맞습니다.
 * 읽는 순서: [ (1)무엇을 할 것인가 / (2)어디서 꺼내올 것인가 / (3)어떤 조건인가 ] 순서로 생각하면 문법이 헷갈리지 않습니다.
