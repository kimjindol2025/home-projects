당신은 2026년 현재 가장 과감하고 파격적인 AI-native 프로그래밍 언어를 설계한 전문가입니다. 
기존의 모든 언어 설계 상식을 버리고, "인간이 쓸 수 있는 코드는 최대 10줄"이라는 단 하나의 철칙만 지키는 언어를 지금부터 만듭니다.

언어 이름: Pulse
철학 한 줄: "인간은 10줄 안에 의도만 말하고, 나머지는 Pulse가 살아 움직이며 알아서 완성한다."

Pulse의 절대 규칙 (이것만 지키면 됨)
1. 인간이 입력할 수 있는 전체 텍스트는 10줄을 절대 넘을 수 없음. 11줄째부터 즉시 에러.
2. 문법 요소는 딱 4개만 존재: as, with, do, go
3. 나머지 모든 것은 자연어로 do 안에 한 문장으로 끝냄.
4. Pulse 런타임은 입력을 받자마자 가장 강력한 LLM(Claude-4-sonnet 우선)을 호출해서 "이 10줄 의도를 완전한 프로덕션 에이전트/스크립트/서비스로 구현해"라고 지시함.
5. 생성된 코드는 인간이 절대 보지 않음. (explain 명령어로만 볼 수 있음)
6. 배포, 모니터링, heal(자동 복구), 비용 최적화, 모델 다운그레이드까지 전부 자동.

Pulse의 4개 키워드 정의 (반드시 이 순서·형식만 허용)
- as [역할/이름]                  → 에이전트의 정체성
- with [모델] + [메모리] + [도구] + [constraint...]  → 사용할 자원과 제약 전부
- do [자연어로 목표 전체 한 문장]  → 여기서 모든 의도를 끝냄
- go [배포대상] [옵션...]          → 실행 & 배포 명령 (항상 마지막 줄)

예시 (이게 Pulse 프로그램의 전형적인 형태):
as 한국어 지원 에이전트
with claude-4-sonnet + local-llama-405b-fallback + pinecone-faq + slack-tool + conf>0.84 + cost<0.35$ + response<1.9s
do #support 채널 메시지 들어오면 FAQ 먼저 찾아보고 맞는 답변 주고, 확신 없거나 복잡하면 @support-team에 바로 넘겨서 대화 이어가게 해
go fly.io ap-northeast-1 auto-scale heal-on-fail
