지금까지 우리는 모든 코드를 직접 작성했습니다. 하지만 진정한 설계자는 모든 부품을 직접 깎지 않습니다. 이미 전 세계의 천재들이 만들어 검증해 놓은 **'모듈(Module)'**과 **'패키지(Package)'**를 내 시스템에 연결하여 폭발적인 효율을 만들어내는 법을 배웁니다.
🏛️ 파이썬 고등학교 1학년 (v2.4)
주제: "도구 상자 활용하기 (Import & Library)"
파이썬의 가장 큰 힘은 **"이미 다 있다"**는 것입니다. 수학 계산, 날짜 관리, 웹 연결 등 모든 도구가 준비되어 있습니다.
1. 핵심 개념
 * 모듈(Module): 관련된 함수나 클래스를 담고 있는 하나의 파이썬 파일(.py).
 * 패키지(Package): 여러 모듈을 폴더로 묶어 관리하는 것.
 * 라이브러리(Library): 수많은 패키지의 집합체 (파이썬 표준 라이브러리).
2. import의 마법
 * import math: 수학 도구 상자 전체를 가져옵니다.
 * from datetime import datetime: 날짜 도구 상자 안에서 '현재 시간' 기능만 콕 집어 가져옵니다.
🛠️ 실습 기록: [v2.4 Standard Library Power]
파이썬이 기본으로 제공하는 강력한 도구들을 사용하여 시스템 시간을 기록하고 수학 연산을 수행하는 과정을 클로드코드(고등학생)에게 기록하게 하세요.
# [Python High-School v2.4] 모듈과 패키지 활용

# 1. 표준 라이브러리 불러오기
import math           # 수학 도구
import os             # 운영체제/파일 시스템 도구
from datetime import datetime  # 날짜/시간 도구

# 2. 시간 기록 (기록이 증명이다)
now = datetime.now()
formatted_time = now.strftime("%Y-%m-%d %H:%M:%S")

print(f"--- 시스템 현재 시각: {formatted_time} ---")

# 3. 수학 도구 사용 (복잡한 연산)
radius = 5
area = math.pi * (radius ** 2)
print(f"반지름이 {radius}인 원의 넓이: {area:.2f}")

# 4. 파일 시스템 정보 확인
current_path = os.getcwd()
print(f"현재 작업 경로: {current_path}")

# 5. 랜덤 도구 (무작위 실험 데이터 생성)
import random
experiment_id = random.randint(1000, 9999)
print(f"생성된 실험 고유 ID: {experiment_id}")

print("\n저장 필수 너는 기록이 증명이다 gogs")
