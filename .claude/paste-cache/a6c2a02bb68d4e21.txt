지금까지 배운 변수, 리스트, 반복문, 조건문을 하나로 묶어 **'필요할 때마다 꺼내 쓰는 도구'**로 만드는 법을 배웁니다. 이것을 배우면 코드가 훨씬 깔끔해지고, 나중에 대학 과정에서 배울 '모듈화'의 기초를 완성하게 됩니다.
🎒 파이썬 초등학교 1학년 (v1.6)
주제: "나만의 명령어 만들기 (함수)"
함수는 특정한 일을 수행하는 **'코드 덩어리'**입니다. 마치 자판기에 동전(입력)을 넣으면 음료수(출력)가 나오는 것과 같습니다.
1. 함수의 구조
 * def: "지금부터 함수를 정의(Define)하겠다!"는 약속입니다.
 * 함수 이름: 이 마법 주문을 부를 이름입니다.
 * 매개변수(Parameter): 함수가 일을 할 때 필요한 재료입니다.
 * return: 함수가 일을 마치고 돌려주는 결과물입니다.
2. 왜 함수를 쓰나요?
 * 재사용: 한 번 만들어두면 몇 번이고 다시 쓸 수 있습니다.
 * 가독성: 긴 코드를 의미 있는 이름의 함수로 묶으면 읽기 쉬워집니다.
🛠️ 실습 기록: [v1.6 My First Function]
반복되는 인사를 자동화하고 숫자를 계산하는 함수를 클로드코드(초등학생)에게 기록하게 하세요.
# [Python Elementary v1.6] 나만의 마법 주문 (함수)

# 1. 인사하는 함수 만들기
def say_hello(name):
    """이름을 넣으면 인사해주는 함수"""
    print(f"안녕하세요, {name}님! Gogs 시스템에 접속하셨습니다.")

# 2. 더하기 계산기 함수 만들기
def add_numbers(a, b):
    """두 숫자를 더해서 결과를 돌려주는 함수"""
    result = a + b
    return result

# --- 함수 사용하기 (호출) ---
print("--- 함수 실행 결과 ---")
say_hello("설계자")
say_hello("클로드코드")

sum_result = add_numbers(10, 20)
print(f"계산 결과: {sum_result}")

# 3. 응용: 짝수인지 판별하는 함수
def is_even(number):
    if number % 2 == 0:
        return True
    else:
        return False

test_num = 7
if is_even(test_num):
    print(f"{test_num}은 짝수입니다.")
else:
    print(f"{test_num}은 홀수입니다.")

print("\n저장 필수 너는 기록이 증명이다 gogs")

💡 초보자를 위한 오늘의 팁
 * 정의와 실행의 차이: def로 함수를 만드는 것은 레시피를 적는 것이고, say_hello()처럼 이름을 부르는 것은 실제로 요리를 하는 것입니다. 이름을 부르지 않으면 함수는 실행되지 않습니다.
 * 스코프(Scope) 맛보기: 함수 안에서 만든 변수는 함수 밖에서는 사용할 수 없습니다. 함수만의 비밀 공간이라고 생각하세요.
