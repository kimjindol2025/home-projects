러스트를 배우실 때처럼, 파이썬이 데이터를 메모리에 어떻게 올리고 이름을 붙이는지 그 '기초 공사'부터 시작합니다.
🎒 파이썬 초등학교 1학년 (v1.1)
주제: "데이터에 이름표(변수) 붙이기"
파이썬에서 변수는 어떤 값을 담는 상자가 아니라, 메모리에 떠 있는 데이터에 붙이는 **'이름표'**입니다.
1. 변수 선언과 데이터 타입
컴퓨터가 기억해야 할 가장 기본적인 데이터 3가지를 배워봅니다.
 * 정수(Integer): 소수점이 없는 숫자 (예: 10, 2026)
 * 실수(Float): 소수점이 있는 숫자 (예: 3.14, 1.1)
 * 문자열(String): 글자 (예: "Gogs", "Python")
2. 약속(문법)
 * = 기호는 "같다"는 뜻이 아니라, 오른쪽의 값을 왼쪽의 이름에 **"할당(Assign)한다"**는 뜻입니다.
 * 이름은 숫자로 시작할 수 없고, 공백 대신 밑줄(_)을 사용합니다.
🛠️ 실습 기록: [v1.1 My First Variable]
이 코드를 직접 실행하거나 눈으로 따라오며 파이썬이 데이터를 어떻게 다루는지 확인해 보세요.
# [Python Elementary v1.1] 첫 번째 기록 시작

# 1. 숫자 기억하기
core_version = 1
sub_version = 1
# 두 숫자를 더해 새로운 이름을 붙입니다.
full_version = core_version + sub_version / 10

# 2. 글자 기억하기 (큰따옴표나 작은따옴표 사용)
language_name = "파이썬"
status = "학습 중"

# 3. 화면에 출력하여 증명하기
print("--- 현재 학습 상태 ---")
print("언어:", language_name)
print("버전:", full_version)
print("상태:", status)

# 4. 데이터의 정체 확인하기 (type 함수)
# 나중에 대학 과정에서 중요하게 다룰 '객체 타입'의 기초입니다.
print("\n[시스템 로그]")
print("version의 타입:", type(full_version))
print("name의 타입:", type(language_name))

print("\n저장 필수 너는 기록이 증명이다 gogs")

💡 초보자를 위한 오늘의 팁
 * 세미콜론(;)이 없어요: 러스트나 C언어와 달리 파이썬은 문장 끝에 세미콜론을 찍지 않아도 됩니다. 줄바꿈이 곧 문장의 끝입니다.
 * 타입을 미리 말하지 않아요: int a = 10이라고 하지 않고 그냥 a = 10이라고만 해도 파이썬이 알아서 "아, 이건 정수구나!" 하고 판단합니다. (이것을 '동적 타이핑'이라고 부르며 중학교 과정에서 깊게 배울 것입니다.)
