🏫 파이썬 중학교 1학년 (v1.7)
주제: "번호표 대신 '이름'으로 데이터 관리하기"
리스트는 0, 1, 2라는 번호표(인덱스)로 데이터를 찾았습니다. 하지만 데이터가 수만 개라면 몇 번이 누구인지 기억하기 힘들겠죠? 딕셔너리는 현실의 사전처럼 **'키(Key)'**와 **'값(Value)'**을 쌍으로 묶어 관리합니다.
1. 딕셔너리의 구조
 * 중괄호 { }: 딕셔너리를 만드는 기호입니다.
 * 키(Key): 데이터를 찾기 위한 이름 (사전의 단어).
 * 값(Value): 실제 데이터 (사전의 뜻).
 * 형태: { "이름": "Gogs", "나이": 20 }
2. 왜 딕셔너리를 쓰나요?
 * 빠른 검색: 번호를 몰라도 "이름"만 알면 즉시 데이터를 찾을 수 있습니다.
 * 의미 있는 데이터: 숫자로 된 번호표보다 데이터의 의미가 명확해집니다.
🛠️ 실습 기록: [v1.7 My First Dictionary]
학생의 정보를 관리하는 시스템을 클로드코드(중학생)에게 기록하게 하세요.
# [Python Middle v1.7] 데이터 사전 (Dictionary)

# 1. 정보를 담은 딕셔너리 만들기
user_info = {
    "name": "설계자",
    "level": "중학생",
    "major": "시스템 공학",
    "point": 100
}

# 2. 데이터 꺼내기 (키 값을 사용)
print("--- 사용자 정보 조회 ---")
print(f"이름: {user_info['name']}")
print(f"전공: {user_info['major']}")

# 3. 새로운 정보 추가 및 수정
user_info["point"] = 150        # 수정
user_info["is_active"] = True  # 추가
print(f"\n[업데이트] 포인트: {user_info['point']}")
print(f"[업데이트] 활동 상태: {user_info['is_active']}")

# 4. 딕셔너리의 열쇠(Key)들만 모아보기
print("\n--- 저장된 정보 항목 리스트 ---")
for key in user_info.keys():
    print(f"- 항목명: {key}")

print("\n저장 필수 너는 기록이 증명이다 gogs")

💡 중학생을 위한 핵심 팁
 * 키(Key)는 유일해야 합니다: 한 사전에 똑같은 단어가 두 번 나올 수 없듯이, 딕셔너리에서도 똑같은 키를 두 번 쓰면 나중에 쓴 값이 이전 값을 덮어버립니다.
 * 가변성(Mutable): 리스트와 마찬가지로 딕셔너리도 자유롭게 내용을 바꾸고 추가할 수 있습니다.
