+# KimSearch v2 - AI 빠른 이해 가이드
+
+## 핵심
+**Gogs 저장소의 3,972개 파일을 검색·분석·비교할 수 있는 REST API 서버**
+
+- 🌐 배포: https://kimsearch-v2.dclub.kr
+- 📊 데이터: 2,167개 파일 / 332,956줄 / 14개 언어
+- ⚡ 상태: 완전 작동 중
+
+---
+
+## AI용 3가지 핵심 API (포트 40007)
+
+### 1. 비슷한 코드 검색
+```
+GET /similar?q=<keyword>&lang=<언어>&limit=<개수>
+
+예:
+  /similar?q=async%20function&lang=js&limit=5
+  /similar?q=database%20connection&lang=py&limit=3
+
+응답:
+{
+  "query": "async function",
+  "lang": "js",
+  "results": [
+    {
+      "repo": "kim",
+      "file": "kim/repo/file.js",
+      "line": 42,
+      "preview": "async function fetchData() {...}",
+      "fullPath": "/tmp/kimsearch-v2/repos-extract/kim/repo/file.js"
+    }
+  ],
+  "count": 5
+}
+```
+
+### 2. 파일 미리보기 (첫 N줄)
+```
+GET /preview?file=<경로>&lines=<줄수>
+
+예:
+  /preview?file=kim/repo/main.js&lines=20
+  /preview?file=kim/repo/README.md
+
+응답:
+{
+  "file": "kim/repo/main.js",
+  "language": "javascript",
+  "totalLines": 450,
+  "previewLines": 20,
+  "preview": "// 첫 20줄의 코드...",
+  "size": 12340,
+  "metadata": {
+    "modified": "2026-02-26T10:00:00.000Z",
+    "bytes": 12340
+  }
+}
+```
+
+### 3. 저장소 정보 & Gogs 이동
+```
+GET /repo?name=<저장소>&file=<파일경로선택>
+
+예:
+  /repo?name=kim
+  /repo?name=kim&file=src/main.js
+
+응답:
+{
+  "name": "kim",
+  "owner": "kim",
+  "path": "/repos-extract/kim",
+  "files": 3972,
+  "size": 23474024,
+  "sizeHuman": "22MB",
+  "gogs": {
+    "url": "https://gogs.dclub.kr/kim",
+    "clone": "https://gogs.dclub.kr/kim.git",
+    "ssh": "ssh://kim@gogs.dclub.kr:2223/kim.git"
+  },
+  "topFiles": [
+    { "lines": 188742, "path": "stats/_continuous_distns.py" },
+    { "lines": 90487, "path": "llama.cpp/convert_hf_to_gguf.py" }
+  ]
+}
+```

+## 기본 기능 3가지 (포트 40006)
+
+### 1. 파일 다운로드
+```
+GET /download?file=<경로>
+→ 파일 컨텐츠 직접 반환 (attachment)
+```
+
+### 2. 파일 비교 (Diff)
+```
+GET /compare?file1=<경로1>&file2=<경로2>
+→ 라인별 diff: added/removed/modified/same 분류
+```
+
+### 3. 통계
+```
+GET /stats
+→ 저장소별, 언어별, 상위 저장소 통계
+```
+
+---
+
+## AI 워크플로우 (추천)
+
+```
+1️⃣ 검색
+   GET /similar?q=패턴&lang=python
+   → 10개 결과 획득
+
+2️⃣ 각 결과 미리보기 (첫 20줄만)
+   GET /preview?file=kim/repo1/file.py&lines=20
+   → 헤더/임포트/기본구조 파악
+
+3️⃣ 전체 파일 다운로드
+   GET /download?file=kim/repo1/file.py
+   → 전체 코드 분석
+
+4️⃣ 좋은 패턴 찾으면 저장소로 이동
+   GET /repo?name=kim/repo1
+   → Gogs 클론 또는 참고 URL 제공
+```
+

+## 기술 스택
+- Node.js (Express 없음, 순수 http 모듈)
+- Grep 기반 패턴 매칭 (빠름)
+- 동기식 처리 (blocking OK, 소규모)
+- 메모리: ~50MB (인덱싱 필요 없음)

+## 테스트 커맨드
+
+### 포트 40007 (AI API)
+```bash
+# 1. 헬스 체크
+ssh ssh253 "curl -s http://localhost:40007/health | jq ."
+
+# 2. 검색
+ssh ssh253 "curl -s 'http://localhost:40007/similar?q=function&lang=js&limit=3' | jq '.results[]'"
+
+# 3. 미리보기
+ssh ssh253 "curl -s 'http://localhost:40007/preview?file=kim/compiler-library-mapper/test_nanoid_noTypes.js&lines=10' | jq ."
+
+# 4. 저장소
+ssh ssh253 "curl -s 'http://localhost:40007/repo?name=kim' | jq '.gogs'"
+```
+
+### 포트 40006 (기본 기능)
+```bash
+# 1. 다운로드
+ssh ssh253 "curl -s 'http://localhost:40006/download?file=kim/file.js' | head -20"
+
+# 2. 비교
+ssh ssh253 "curl -s 'http://localhost:40006/compare?file1=kim/file1.py&file2=kim/file2.py' | jq '.summary'"
+
+# 3. 통계
+ssh ssh253 "curl -s http://localhost:40006/stats | jq '.total'"
+```
+
+### 외부 접속 (도메인)
+```bash
+curl https://kimsearch-v2.dclub.kr/stats | jq .
+curl https://kimsearch-v2.dclub.kr/health | jq .
+```
