파이썬 고등학교 1학년의 세 번째 수업, [v2.3: 상속(Inheritance) — 설계도의 진화와 재사용] 단계입니다.
우리가 만든 GogsRobot 설계도는 훌륭하지만, 특정 분야에 특화된 로봇(예: 비행 로봇, 수중 로봇)이 필요할 때마다 처음부터 다시 설계하는 것은 비효율적입니다. 파이썬의 상속 기능을 사용하면 기존 설계도의 기능을 그대로 물려받으면서 새로운 기능만 덧붙일 수 있습니다.
🏛️ 파이썬 고등학교 1학년 (v2.3)
주제: "부모의 유산을 이어받아 더 강력하게 만들기"
상속은 부모 클래스(Parent Class)의 모든 속성과 메서드를 자식 클래스(Child Class)가 물려받는 것입니다.
1. 상속의 핵심 개념
 * 재사용: 똑같은 코드를 다시 쓸 필요가 없습니다.
 * 확장: 새로운 기능을 추가하거나(Method Addition), 부모의 기능을 내 방식대로 고쳐 쓸 수 있습니다(Method Overriding).
 * super(): "부모님, 도와주세요!" 부모 클래스의 기능을 그대로 가져와서 쓰고 싶을 때 사용합니다.
2. 왜 고등 과정인가요?
 * 시스템이 거대해질수록 공통된 기능을 하나로 묶고 특수 기능만 분리하는 '계층적 설계' 능력이 필수적이기 때문입니다.
🛠️ 실습 기록: [v2.3 Evolution of Robot]
기존 GogsRobot을 상속받아 비행 기능이 추가된 FlyingGogsRobot을 만드는 과정을 클로드코드(고등학생)에게 기록하게 하세요.
# [Python High-School v2.3] 상속 (Inheritance)

# 1. 부모 클래스 (v2.2에서 만든 기초 설계도)
class GogsRobot:
    def __init__(self, name):
        self.name = name
        print(f"🤖 [{self.name}] 기본 유닛 생성.")

    def work(self):
        print(f"  [ACTION] {self.name}이(가) 지상 업무를 수행합니다.")

# 2. 자식 클래스 (상속: 괄호 안에 부모 클래스 이름을 넣습니다)
class FlyingGogsRobot(GogsRobot):
    def __init__(self, name, altitude):
        # super()를 사용하여 부모의 __init__을 먼저 호출합니다.
        super().__init__(name)
        self.altitude = altitude
        print(f"🚀 [{self.name}] 비행 모듈 장착 완료. (고도: {self.altitude}m)")

    # 메서드 오버라이딩 (부모의 기능을 내 방식대로 덮어쓰기)
    def work(self):
        print(f"  [ACTION] {self.name}이(가) {self.altitude}m 상공에서 정찰 업무를 수행합니다.")

    # 새로운 기능 추가
    def fly(self):
        print(f"  [FLY] {self.name}이(가) 날아오릅니다!")

# --- 실증 실험 ---
print("--- 로봇 진화 테스트 ---")
basic_bot = GogsRobot("지상봇")
flying_bot = FlyingGogsRobot("스카이봇", 500)

basic_bot.work()   # 지상 업무 수행
flying_bot.work()  # 상공 정찰 업무 수행 (오버라이딩된 기능)
flying_bot.fly()   # 자식 클래스에만 있는 기능

print("\n저장 필수 너는 기록이 증명이다 gogs")

💡 고등학생을 위한 핵심 팁
 * Is-A 관계: 상속은 "자식은 부모의 일종이다(A FlyingRobot is a Robot)"라는 관계가 성립할 때 사용하는 것이 정석입니다.
 * 중복의 제거: 여러 클래스에서 공통으로 쓰는 코드가 있다면, 부모 클래스로 끌어올려 하나로 관리하세요. 그것이 바로 대학 과정에서 배울 **'유지보수성'**의 핵심입니다.
> 저장 필수 너는 기록이 증명이다 gogs
> 
이제 단순한 객체를 넘어 클래스 간의 관계를 설계할 수 있게 되었습니다. 고등학교 과정의 기초 설계 역량이 거의 완성되어 가고 있습니다.
