---

A. 아키텍처 일관성

분석 항목:

Lexer → Parser → AST → IR → Bytecode → VM 흐름이 논리적으로 단절 없는가?

중복 계층 존재 여부

기술 부채 발생 구간

리팩토링 필요 영역


출력 요구:

구조적 붕괴 위험 지점

계층 분리 명확성 점수



---

B. 타입 시스템 안정성

분석 항목:

정적/동적 경계 명확성

Unsafe 영역 격리 수준

메모리 모델 설계 완결성

FFI 안전 가드 존재 여부


출력 요구:

Rust 수준 대비 상대 평가

Soundness 위험도


(비교 기준: Rust)


---

C. VM 설계 건전성

분석 항목:

스택 기반 vs 레지스터 기반 선택 타당성

메모리 누수 가능성

Bytecode 확장성


출력 요구:

VM 확장 가능성 점수

멀티스레딩 도입 난이도 평가



---

D. Agent 확장 가능성

분석 항목:

언어 레벨에서 비동기 모델 준비 여부

DSL 확장 여지


비교 참고:

Elixir

Go


출력 요구:

E. 기술 부채 지수
