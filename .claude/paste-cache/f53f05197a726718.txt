📌 Phase 2 Step-by-Step 지침서 (Const/Load/Store 시작)

Step 1 — Const 연산 Heap 기반 변환

1. VM executeConst(opcode: ConstOp) 수정

// 기존: stack.push(value)
// 수정:
const ptr = heap.allocate(op.value)  // HeapManager에 Value 할당
stack.push(ptr)                       // stack에는 ptr만 보관


2. 테스트

stack top ptr 확인 → heap.get(ptr).val === op.value

기존 Const 테스트 7/7 통과 재확인





---

Step 2 — Load 연산 Heap 기반

1. VM executeLoad(opcode: LoadOp)

const ptr = globals.get(op.name)
stack.push(ptr)  // stack에는 ptr만


2. 테스트

기존 글로벌 값과 heap.get(ptr).val 일치

stack push/pop 연동 테스트





---

Step 3 — Store 연산 Heap 기반

1. VM executeStore(opcode: StoreOp)

const ptr = stack.pop()
const valPtr = stack.pop()
heap.decrement(ptr)             // 이전 값 refCount 감소
heap.set(ptr, heap.get(valPtr)) // 새 값 refCount 증가


2. 테스트

기존 변수 값 변경 확인

refCount 증가/감소 정상 확인

stack과 globals 일치 확인





---

Step 4 — 검증

기존 7개 테스트 재실행

HeapManager refCount, freed count 체크

음수 refCount → 즉시 crash (안전 모드)
