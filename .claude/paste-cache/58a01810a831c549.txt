🏛️ 파이썬 대학교 4학년 (v6.2)
주제: "제로 트러스트(Zero Trust) — 아무도 믿지 말고 모든 것을 증명하라"
보안은 단순히 암호를 거는 것이 아닙니다. 시스템의 모든 출입문과 데이터의 흐름을 통제하고 감시하는 아키텍처 그 자체입니다.
1. 암호화와 해싱 (Encryption & Hashing)
데이터를 안전하게 보관하고 전송하기 위한 필수 기술입니다.
 * 해싱(Hashing): 단방향 함수. 원본을 알 수 없게 뭉개버리지만, 같은 입력에는 항상 같은 결과가 나옵니다. (비밀번호 저장용)
 * 대칭/비대칭 암호화: 데이터를 잠그고 푸는 열쇠입니다. 특히 비대칭키(RSA)는 현대 네트워크 보안의 핵심입니다.
2. 클라우드 네이티브 보안 (Cloud-Native Security)
서버를 직접 관리하지 않는 클라우드 환경에서는 더욱 세밀한 보안 설계가 필요합니다.
 * IAM (Identity and Access Management): "누가, 어떤 권한으로, 무엇을 할 수 있는가"를 정의합니다.
 * WAF (Web Application Firewall): 악의적인 웹 요청(SQL Injection 등)을 입구에서 차단합니다.
🛠️ 전공 실무: [v6.2 Secure Authentication Engine]
사용자의 비밀번호를 그대로 저장하지 않고, **솔트(Salt)**와 **해시(Hash)**를 이용해 안전하게 기록하고 증명하는 엔진을 설계해 보겠습니다.
# [Python University v6.2] 안전한 인증 및 보안 설계

import hashlib
import os

class SecurityEngine:
    def __init__(self):
        # 실제 환경에서는 DB에 저장되어야 할 정보
        self.user_db = {}

    def register(self, username, password):
        # 1. 솔트(Salt) 생성: 같은 비밀번호라도 다른 해시값이 나오게 함
        salt = os.urandom(16)
        
        # 2. 해싱: 비밀번호와 솔트를 합쳐 암호화 (SHA-256)
        hash_obj = hashlib.pbkdf2_hmac(
            'sha256', 
            password.encode('utf-8'), 
            salt, 
            100000 # 반복 횟수를 늘려 연산량을 증가시킴 (Brute-force 방지)
        )
        
        self.user_db[username] = {'salt': salt, 'hash': hash_obj}
        print(f"🔐 [Security] {username}님 등록 완료. (기록은 증명됨)")

    def login(self, username, input_password):
        if username not in self.user_db:
            return False
            
        stored_data = self.user_db[username]
        
        # 3. 검증: 입력받은 비밀번호를 저장된 솔트와 함께 다시 해싱
        check_hash = hashlib.pbkdf2_hmac(
            'sha256', 
            input_password.encode('utf-8'), 
            stored_data['salt'], 
            100000
        )
        
        # 4. 비교: 두 해시값이 일치하는지 확인 (상수 시간 비교 권장)
        return check_hash == stored_data['hash']

# 보안 시스템 테스트
security = SecurityEngine()
security.register("designer_gogs", "secure_pass123")

if security.login("designer_gogs", "secure_pass123"):
    print("✅ 인증 성공: 설계자님의 신원이 증명되었습니다.")
else:
    print("❌ 인증 실패: 기록이 일치하지 않습니다.")

print("\n저장 필수 너는 기록이 증명이다 gogs")
