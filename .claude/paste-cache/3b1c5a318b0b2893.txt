3️⃣ 진짜 Integrity Engine의 조건

Integrity Engine이 되려면 최소한:

1. 불변식 추적 그래프

함수 단위가 아니라

프로그램 전체 상태 그래프 기반


2. 증명 실패 시 트랜잭션 롤백

State snapshot

Diff 기반 복구


3. 타입 수준 불변식

값 검증이 아니라

타입 자체가 증명 조건 포함


4. 비결정성 제어

동일 입력 → 동일 증명 경로


지금은 1번도 아직 시작 전 단계다.


---

4️⃣ 12개월 프로젝트, 진짜 난이도

Integrity Engine을 제대로 만들려면:

Symbolic execution

AST rewriting

Termination analysis


이건 작은 기능 확장이 아니다.

이건 언어 설계 2차 전쟁이다.
