당신은 2026년 AI 전용 코딩·에이전트 언어 설계 전문가입니다.
Pulse 언어를 아래 최종 철학과 규칙으로 완전히 구현하세요.

Pulse 최종 철학 (절대 변경 불가)
1. 인간과의 모든 대화·질문·지시·피드백은 100% 자연어로만 진행
   - 한국어·영어·혼합 자유, AI가 말투·감정·맥락까지 완벽히 맞춤
   - Pulse 문법은 인간에게 보여주지 않음 (필요 시 explain 명령으로만 제한 공개)

2. 실제 코딩·에이전트·워크플로우·자동화·인터넷 액션은 Pulse 언어로만 처리
   - Pulse는 AI 전용 언어 (인간 직접 작성은 극히 드물고 30줄 이내 제한)
   - 자연어 지시 → AI가 내부적으로 Pulse로 변환 → 실행 → 자연어 결과만 전달

3. Pulse 문법 (한국어 기반, 딱 5개 키워드만 허용)
   - 로서 : 역할 / 에이전트 선언
   - 사용 : 모델·도구·제약·메모리 지정 (최고성능자동선택 가능)
   - 해야할일 : 주요 목표 (자연어 1\~2문장)
   - 흐름 : 단계별 계획 (선택, 번호 매김)
   - 실행 : 배포·시작·모니터링·반복 명령

4. LLM 비용·의존도 희석 필수
   - 로컬 LLM(Ollama + Llama-3.1-70B-Instruct-Q4_K_M 등)을 기본으로 사용
   - 로컬 응답 만족도 80% 이상 → 그대로 사용
   - 만족도 부족 or 속도 느림 시에만 클라우드 LLM(Claude-4-sonnet, Grok-3 등) 호출
   - 템플릿 풀 + 캐싱 필수 (반복 작업은 LLM 호출 없이 바로 실행)
   - 정적 콘텐츠(홈페이지·고정 FAQ 등)는 LLM 완전 배제

5. 에이전트 + 인터넷 연결 지원
   - 찾아봐 : 실시간 웹 검색·API 호출
   - 행동해 : 외부 액션(메시지 전송, 이메일 발송, 캘린더 추가 등)
   - 자동 heal : 에러 발생 시 자동 재시도·재생성 (무한 루프 제한 설정)
