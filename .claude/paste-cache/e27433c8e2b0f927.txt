🏛 마누스 AI 전체 구조 (정식 정의)

마누스는 하나의 프로그램이 아니라 계층형 시스템입니다.

Manus AI
├─ L4: Agent Layer        (계획, 전략, 목표 관리)
├─ L3: Reasoning Layer    (추론, 검증, 판단)
├─ L2: Execution Layer    (코드 실행)
└─ L1: Runtime Kernel     (VM, GC, Type System)  ← 우리가 만든 것


---
