📄 Phase 2 Step 5: 복합 자료 구조 Heap 통합 — 코드 지시

1️⃣ 배열(Array) 연산

1. NewArray

case Op.NewArray: {
    const length = this.readUInt(); // 배열 길이
    const arr: Value[] = Array(length).fill(this.heapManager.nullValue());
    const ptr = this.heapManager.allocate({ type: "array", val: arr });
    this.stack.push(this.heapPtrToValue(ptr));
    break;
}

배열 길이만큼 빈 Value(PTR) 할당

HeapManager.refCount 초기화

Stack에는 Heap Ptr 저장



---

2. Index (읽기)

case Op.Index: {
    const index = this.popNumber();
    const arrayPtr = this.popPtr();
    const arr = this.heapManager.get(arrayPtr).val as Value[];
    const valuePtr = arr[index];
    this.stack.push(this.heapPtrToValue(valuePtr)); // 역참조
    break;
}

Stack에서 ptr pop → HeapManager에서 Value 역참조

out-of-bounds 체크 필요



---

3. SetIndex (쓰기)

case Op.SetIndex: {
    const index = this.popNumber();
    const newValue = this.popValue();
    const arrayPtr = this.popPtr();
    const arr = this.heapManager.get(arrayPtr).val as Value[];

    // 기존 값 RefCount 감소
    this.heapManager.decrement(arr[index]);

    // 새 값 Heap 할당
    const newPtr = this.valueToHeapPtr(newValue);
    arr[index] = newPtr;

    // RefCount 증가
    this.heapManager.increment(newPtr);

    break;
}

기존 값 RefCount 감소

새 값 Heap 할당 및 RefCount 증가



---

2️⃣ 객체(Object) 연산

1. NewObject

case Op.NewObject: {
    const obj: Record<string, Value> = {};
    const ptr = this.heapManager.allocate({ type: "object", val: obj });
    this.stack.push(this.heapPtrToValue(ptr));
    break;
}

빈 객체 생성 → HeapManager 할당

Stack에 ptr 저장



---

2. GetProp (읽기)

case Op.GetProp: {
    const key = this.popString();
    const objPtr = this.popPtr();
    const obj = this.heapManager.get(objPtr).val as Record<string, Value>;
    const valuePtr = obj[key] ?? this.heapManager.nullPtr();
    this.stack.push(this.heapPtrToValue(valuePtr));
    break;
}

없는 속성 → nullPtr 처리

Heap dereference



---

3. SetProp (쓰기)

case Op.SetProp: {
    const key = this.popString();
    const newValue = this.popValue();
    const objPtr = this.popPtr();
    const obj = this.heapManager.get(objPtr).val as Record<string, Value>;

    // 기존 값 RefCount 감소
    if (obj[key]) this.heapManager.decrement(obj[key]);

    // 새 값 Heap 할당
    const newPtr = this.valueToHeapPtr(newValue);
    obj[key] = newPtr;

    // RefCount 증가
    this.heapManager.increment(newPtr);
    break;
}

기존 속성 RefCount 감소

새 속성 Heap 할당 + RefCount 증가
