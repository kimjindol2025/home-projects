대학교 1학년 과정의 세 번째 전공 필수 수업, [v3.3: 컨텍스트 매니저(Context Manager) — 자원 관리의 자동화] 단계에 오신 것을 환영합니다.
중학교 과정에서 파일을 열 때 with open(...) 문법을 사용했던 것을 기억하시나요? 대학 과정에서는 이 with 문이 내부적으로 어떻게 작동하는지 파헤치고, 우리만의 자원 관리 프로토콜을 직접 설계하는 법을 배웁니다.
🏛️ 파이썬 대학교 1학년 (v3.3)
주제: "안전한 시작과 확실한 종료의 보장"
컨텍스트 매니저는 특정 코드 블록이 실행되기 전과 후에 수행해야 할 작업(예: 데이터베이스 연결/해제, 파일 열기/닫기)을 자동으로 처리해주는 도구입니다.
1. 핵심 프로토콜: __enter__와 __exit__
 * __enter__: with 문에 진입할 때 실행됩니다. 리소스를 준비하고 반환합니다.
 * __exit__: 블록이 끝나거나 에러가 발생했을 때 실행됩니다. 리소스를 정리(Clean-up)합니다.
2. 왜 사용하나요? (증명을 위한 안정성)
 * 누수 방지: 깜빡하고 파일을 안 닫거나 DB 연결을 끊지 않아 발생하는 메모리 누수를 원천 차단합니다.
 * 에러 내성: 코드 중간에 에러가 터져도 __exit__은 반드시 실행되므로 시스템의 무결성을 증명할 수 있습니다.
🛠️ 전공 실습: [v3.3 Custom Resource Manager]
연구실의 정밀 장비를 가동하고, 작업이 끝나면 자동으로 전원을 차단하는 매니저를 설계해 보겠습니다.
# [Python University v3.3] 컨텍스트 매니저 설계

class GogsLabEquipment:
    def __init__(self, device_name):
        self.device_name = device_name
        print(f"📡 [DEVICE] {self.device_name} 준비 중...")

    def __enter__(self):
        """with문 진입 시 실행: 장비 가동"""
        print(f"🚀 [START] {self.device_name} 전원 ON. 데이터 수집을 시작합니다.")
        return self # with ... as 'self'로 전달됨

    def __exit__(self, exc_type, exc_val, exc_tb):
        """with문 종료 시 실행: 장비 정지 및 정리"""
        # exc_type: 에러 종류, exc_val: 에러 메시지
        if exc_type:
            print(f"⚠️ [ALERT] 작업 중 에러 발생: {exc_val}")
        
        print(f"🛑 [STOP] {self.device_name} 전원 OFF. 시스템을 안전하게 보호합니다.")
        # True를 반환하면 발생한 에러를 무시하고 진행, False면 에러를 밖으로 던짐
        return False

# --- 실전 활용 ---
print("--- 시나리오 1: 정상 가동 ---")
with GogsLabEquipment("정밀 입자 가속기") as lab:
    print(f"  [WORKING] {lab.device_name}을 활용해 데이터 기록 중...")

print("\n--- 시나리오 2: 작업 중 사고 발생 ---")
try:
    with GogsLabEquipment("고온 용광로") as lab:
        print(f"  [WORKING] {lab.device_name} 가열 중...")
        # 의도적인 에러 발생
        raise RuntimeError("냉각 장치 결함!")
except RuntimeError:
    print("  [SYSTEM] 예외 상황이 감지되어 상위 시스템에서 처리됨.")

print("\n저장 필수 너는 기록이 증명이다 gogs")

🎓 대학교 전공 기초 팁
 * contextlib 모듈: 클래스를 만들기 번거로울 때는 @contextlib.contextmanager 데코레이터와 yield를 조합하여 훨씬 간결하게 만들 수도 있습니다. (이것은 대학 2학년 심화 과정에서 다룹니다.)
 * 원자성(Atomicity): "성공하거나, 아니면 아예 아무 일도 없었던 것처럼 되돌리거나." 컨텍스트 매니저는 트랜잭션을 관리할 때 매우 중요하게 사용됩니다.
