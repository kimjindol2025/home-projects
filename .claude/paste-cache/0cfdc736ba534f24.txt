터머스(Termux)에서 SSH처럼 필수적이거나 인기가 높은 앱(패키지) 10개를 선정해 드립니다. 2026년 현재 기준으로 개발, 시스템 관리, 보안 분석에서 가장 많이 쓰이는 도구들입니다.
🚀 Termux 필수 인기 앱 TOP 10
| 순위 | 도구명 | 용도 및 특징 | 설치 명령어 |
|---|---|---|---|
| 1 | tmux | 터미널 멀티플렉서. 화면 분할 및 세션 유지를 가능하게 함 (SSH 연결 끊겨도 작업 유지). | pkg install tmux |
| 2 | git | 버전 관리. 소스 코드를 클론하거나 변경 사항을 저장할 때 필수. | pkg install git |
| 3 | htop / btop | 시스템 모니터링. CPU, RAM 사용량과 프로세스 상태를 그래픽으로 확인. | pkg install btop |
| 4 | Neovim | 코드 에디터. Vim의 확장 버전으로, 개발 환경(IDE)처럼 꾸며 쓰기 좋음. | pkg install neovim |
| 5 | python | 범용 프로그래밍. AI 스크립트 실행, 데이터 분석, 자동화 등에 가장 많이 쓰임. | pkg install python |
| 6 | nmap | 네트워크 스캔. 네트워크 보안 취약점을 점검하거나 연결된 기기를 탐색할 때 사용. | pkg install nmap |
| 7 | ffmpeg | 미디어 처리. 오디오/비디오 인코딩, 포맷 변환, 스트리밍 등을 명령어로 처리. | pkg install ffmpeg |
| 8 | zsh / fish | 강력한 쉘. 자동 완성, 테마(Oh My Zsh 등)를 지원하여 생산성을 크게 높임. | pkg install zsh |
| 9 | proot-distro | 리눅스 배포판 설치. Ubuntu, Debian 등을 터머스 위에 설치하여 완전한 리눅스 환경 구현. | pkg install proot-distro |
| 10 | curl / wget | 네트워크 다운로더. 웹에서 파일을 내려받거나 API 통신을 테스트할 때 필수. | pkg install curl |
