3. 상수와 iota (Constants & Enumeration)
상수는 값이 변하지 않는 데이터를 정의할 때 사용합니다. 특히 Go의 iota는 열거형(Enum)을 구현할 때 매우 우아한 방식을 제공합니다.
const (
    Freshman = iota // 0
    Sophomore       // 1
    Junior          // 2
    Senior          // 3
)

iota는 상수가 선언될 때마다 0부터 1씩 자동으로 증가하며 대입됩니다. 시스템의 상태(State)를 관리할 때 필수적입니다.
🔬 전공 심화: 제로 값 (Zero Values)
Go는 변수를 선언하고 초기화하지 않았을 때, 쓰레기 값이 아닌 **'기본값(Zero Value)'**을 강제로 할당합니다. 이는 메모리 안전성을 보장하는 핵심 장치입니다.
 * int: 0
 * float: 0.0
 * bool: false
 * string: "" (빈 문자열)
 * pointer, slice, map, channel: nil
