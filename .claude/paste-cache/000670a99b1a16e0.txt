3️⃣ 핵심 개념 정리

TF (Term Frequency)

해당 chunk 안에서 단어가 몇 번 등장하는가

DF (Document Frequency)

전체 chunk 중 몇 개에 등장하는가

IDF

많이 등장하는 단어는 가치 ↓

IDF = log(총 chunk 수 / (1 + 단어 등장 chunk 수))


---

6️⃣ 점수 계산

간단 BM25에 가까운 형태:

function score(queryTerms, chunk, index, avgLen) {
  let score = 0;
  const chunkLen = chunk.termCount;

  for (const term of queryTerms) {
    const entry = index[term];
    if (!entry) continue;

    const tf = chunk.tfMap[term] || 0;
    const idf = Math.log(totalChunks / (1 + entry.df));

    const k = 1.5;
    const b = 0.75;

    const norm = tf * (k + 1) /
      (tf + k * (1 - b + b * (chunkLen / avgLen)));

    score += idf * norm;
  }

  return score;
}

이제:

짧은 chunk 과대평가 방지

흔한 단어 자동 패널티

많이 등장하는 핵심어 가중
