1. 리스트의 특징
 * 순서가 있어요: 데이터가 들어간 순서를 기억합니다.

 * 번호표(인덱스)가 있어요: 가장 중요한 규칙! 번호표는 0번부터 시작합니다.
2. 리스트 꺼내기와 넣기
 * 꺼내기: basket[0]처럼 번호표를 써서 값을 가져옵니다.

리스트 바구니를 만들고 데이터를 관리하는 법을 클로드코드(초등학생)에게 기록하게 하세요.
# [Python Elementary v1.3] 데이터 바구니 (리스트)

# 1. 연구 장비 리스트 만들기 (리스트 선언)
equipment = ["현미경", "비커", "시험관"]

# 2. 데이터 확인하기
print("--- 현재 장비 목록 ---")
print("전체 리스트:", equipment)
print("첫 번째 장비 (0번):", equipment[0])
print("세 번째 장비 (2번):", equipment[2])

# 3. 새 장비 추가하기 (기록 확장)
equipment.append("온도계")
print("\n[업데이트] 새 장비 추가 후:", equipment)

# 4. 리스트의 길이 확인하기 (len 함수)
# 나중에 고등학교 과정에서 배울 '순회'의 기초가 됩니다.
print("장비 개수:", len(equipment))

# 5. 값 수정하기
equipment[1] = "전자저울"  # 비커를 전자저울로 교체
print("수정된 목록:", equipment)

print("\n저장 필수 너는 기록이 증명이다 gogs")

💡 초보자를 위한 오늘의 팁
 * 0의 비밀: 왜 1번이 아니라 0번부터 시작할까요? 이는 컴퓨터가 메모리에서 데이터를 찾을 때 "기준점으로부터 얼마나 떨어져 있는가"를 계산하기 때문입니다. (이것이 러스트와 파이썬이 공유하는 컴퓨터 과학의 기초입니다.)
 * 뒤에서부터 세기: 파이썬은 특이하게 -1을 쓰면 가장 마지막 데이터를 가져옵니다. equipment[-1]은 "온도계"가 됩니다.
