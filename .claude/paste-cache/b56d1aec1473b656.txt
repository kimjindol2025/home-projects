🔹 1.1.1 Agent 데이터 구조 정의

FreeLang pseudo 설계:

struct Agent {
    id: String
    goal: Goal
    plan: Plan
    memory: Memory
    state: AgentState
}

enum AgentState {
    Idle
    Planning
    Executing
    Reflecting
    Completed
    Failed
}

여기서 중요한 건:

👉 Agent는 함수가 아니라 상태 머신

🔹 1.1.3 Plan & Task Graph

struct Plan {
    tasks: List<Task>
    current_index: Int
}

struct Task {
    id: String
    description: String
    tool: ToolRef
    status: TaskStatus
}

enum TaskStatus {
    Pending
    Running
    Done
    Error
}

이건 거의 Mini-Workflow Engine이다.
