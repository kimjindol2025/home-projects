1. 배포 완료! 분산 은행 시스템이 설치되었습니다.

2. Pod 상태 확인:
   kubectl get pods -n {{ .Values.namespace.name }} -o wide
   kubectl get pods -n {{ .Values.namespace.name }} -w

3. 로그 확인:
   kubectl logs -f deployment/{{ include "distributed-bank.fullname" . }} -n {{ .Values.namespace.name }}

4. Pod 진입 (디버깅용):
   kubectl exec -it <pod-name> -n {{ .Values.namespace.name }} -- /bin/bash

5. 포트 포워딩:
   {{- if eq .Values.service.external.type "LoadBalancer" }}
   kubectl port-forward svc/{{ include "distributed-bank.fullname" . }}-lb {{ .Values.service.external.ports.http }}:{{ .Values.service.internal.ports.http }} -n {{ .Values.namespace.name }}
   {{- else }}
   kubectl port-forward svc/{{ include "distributed-bank.fullname" . }} {{ .Values.service.internal.ports.http }}:{{ .Values.service.internal.ports.http }} -n {{ .Values.namespace.name }}
   {{- end }}

6. 서비스 정보:
   {{- if eq .Values.service.external.type "LoadBalancer" }}
   EXTERNAL_IP=$(kubectl get svc {{ include "distributed-bank.fullname" . }}-lb -n {{ .Values.namespace.name }} -o jsonpath='{.status.loadBalancer.ingress[0].ip}')
   echo "LoadBalancer IP: $EXTERNAL_IP"
   {{- end }}

7. 헬스체크:
   {{- if eq .Values.service.external.type "LoadBalancer" }}
   curl http://$EXTERNAL_IP:{{ .Values.service.external.ports.http }}/health
   {{- else }}
   kubectl port-forward svc/{{ include "distributed-bank.fullname" . }} 8080:{{ .Values.service.internal.ports.http }} -n {{ .Values.namespace.name }} &
   curl http://localhost:8080/health
   {{- end }}

8. 메트릭 확인:
   kubectl port-forward svc/{{ include "distributed-bank.fullname" . }} {{ .Values.debug.metricsPort }}:{{ .Values.debug.metricsPort }} -n {{ .Values.namespace.name }}
   curl http://localhost:{{ .Values.debug.metricsPort }}/metrics

9. 배포 상태:
   kubectl rollout status deployment/{{ include "distributed-bank.fullname" . }} -n {{ .Values.namespace.name }}

10. 차트 언설치:
    helm uninstall {{ .Release.Name }} -n {{ .Values.namespace.name }}
